Digital Logic Design

Logisim Digital Circuits Simulation
February 2023 - March 2023
University Lab Project

Project Overview

Designed and simulated a functional digital logic circuit using Logisim, showcasing proficiency in digital logic design principles. Employed various components like logic gates and multiplexers for circuit implementation and simulation accuracy.

Technical Implementation

  • Used AND, OR, NOT, XOR gates to build core logic functions
  • Integrated multiplexers to manage data routing
  • Simulated logic circuits to test truth tables and performance
  • Organized logic design hierarchically using Logisim subcircuits

Results & Impact

  • Accurate simulation of combinational and sequential circuits
  • Strengthened understanding of circuit design concepts
  • Optimized layout for educational presentations
