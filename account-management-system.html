Account Management System

Java Authentication Desktop Application
August 2023
Academic Project

Project Overview

Created a desktop-based Account Management System in Java, designed to manage user accounts with secure login, balance tracking, and automated credential generation.

Technical Implementation

  • Java Swing used for the graphical user interface
  • Random ID and password generator for new users
  • Login system with input validation and error handling
  • Balance update, view, and basic transaction support

Results & Impact

  • Improved understanding of Java object-oriented programming
  • Applied exception handling and input validation practices
  • Demonstrated real-world account handling logic
