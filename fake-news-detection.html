Data Science Project

Project Overview

This project implements a fake news detection system by applying K-means clustering on news articles to identify groups of similar content that might indicate misinformation.

Technical Implementation

  • Collected and preprocessed textual news data including tokenization and feature extraction
  • Applied TF-IDF vectorization to convert text data into numerical features
  • Implemented K-means clustering to group articles based on content similarity
  • Analyzed clusters to detect potential fake news patterns
  • Evaluated clustering performance using metrics like silhouette score

Results & Impact

  • Identified clusters with high probability of containing fake news articles
  • Provided insights into misinformation spread patterns
  • Demonstrated practical application of unsupervised learning in NLP
