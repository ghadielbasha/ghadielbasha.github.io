Project Overview

Built a deep learning system to classify images of cats and dogs using Convolutional Neural Networks (CNNs). The model was trained on thousands of labeled images to automatically detect and learn key features distinguishing the two animal types.

Technical Implementation

  • Used TensorFlow and Keras to build a CNN architecture
  • Preprocessed the image dataset: resizing, normalization, and data augmentation
  • Implemented convolutional layers followed by pooling, flattening, and dense layers
  • Used binary cross-entropy loss and accuracy metrics during training
  • Evaluated model on validation set and optimized hyperparameters

Results & Impact

  • Achieved high classification accuracy (over 90%)
  • Learned how to build and fine-tune deep learning models
  • Gained experience with image-based datasets and CNN architectures
